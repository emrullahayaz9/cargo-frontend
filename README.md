1. Clone the repository `git clone https://github.com/emrullahayaz9/cargo-frontend.git`
2. open terminal and write `cd ./cargo-frontend`, then click enter.
3. run `npm install`
4. run `npm start`

## Ensure backend server is running.
## Ensure this frontend is not running on port 3000. (Grafana will be up on port 3000)
